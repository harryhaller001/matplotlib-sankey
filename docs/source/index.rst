=================
matplotlib-sankey
=================

Generate sankey plot with matplotlib. For an example, see :doc:`example`.


Get started
-----------

You can install ``matplotlib-sankey`` with ``pip``::

    pip install matplotlib-sankey


For more details, see :ref:`install`.


Contact
-------
